# Imágenes de consolas para el Lobby

Opcional: podés agregar imágenes de cada consola para que aparezcan en las tarjetas del selector de edición.

Archivos esperados:
- `gb.png` — Game Boy
- `genesis.png` — Sega Genesis / Mega Drive
- `gba.png` — Game Boy Advance

Si no existen, las tarjetas muestran solo el nombre. Cualquier imagen PNG funciona; se escalará automáticamente.
